<Heading hidden>SQL Generation</Heading>

We support SQL generation in these resources: PostgreSQL, MySQL, Oracle DB, MariaDB, TIDB, SupabaseDB, and ClickHouse. You can enter the natural language and get a ready-to-run SQL statement when using these resources. 

**Enter the app edit page and create an action using one of the resources above.**

**Select the statement type you want to generate**
