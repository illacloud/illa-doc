## <h2 hidden>Integration List</h2>

ILLA offers a range of pre-built integrations that allow you to easily connect to your data sources with minimal configuration. Simply select the appropriate integration from the list of available options, enter the necessary credentials, and you can start using your data source within ILLA. This eliminates the need to set up complex configurations and enables you to quickly start working with your data.

### Database integrations

| Name | Description |
| --- | --- |
| Mysql  | MySQL is an open-source relational database management system (RDBMS) that is based on the Structured Query Language (SQL). |
| PostgreSQL | PostgreSQL is an open-source object-relational database management system (ORDBMS) that is known for its reliability and feature-richness. |
| MariaDB | MariaDB is an open-source relational database management system that is a fork of the MySQL database. |
| TiDB | TiDB is an open-source NewSQL database that supports Hybrid Transactional and Analytical Processing workloads. |
| Redis | Redis is an open-source in-memory data structure store that is often used as a database, cache, and message broker. |
| MongoDB | MongoDB is a free and open-source cross-platform document-oriented database program. |
| Elastic Search | Elasticsearch is an open-source, distributed search and analytics engine built on top of Apache Lucene. |
| Firebase | Firebase is a cloud-based mobile and web application development platform developed by Google. |
| Supabase DB | Supabase is an open-source platform for building real-time serverless applications. It is built on top of PostgreSQL and provides a number of tools and services for quickly building and deploying serverless applications. |
| Clickhouse | ClickHouse is an open-source column-oriented database management system that is designed for online analytical processing (OLAP) and real-time data analysis. |
