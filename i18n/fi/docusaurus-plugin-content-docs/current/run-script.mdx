#### setFilters(esineiden valikoima, merkkijono)

Käytetään suodattimien asettamiseen.

**Ensimmäinen parametri** on joukko esineitä. Jokainen objekti edustaa suodatuksen ehtoa. Se sisältää kolme ominaisuutta:

**\<1>** **`id`** : Tämä viittaa tämän sarakkeen tunnisteeseen. Jos tämä sarake on tulos tietokantakyselystä, niin tunnus olisi sarake nimi, jonka olet hakenut kyselystä. Esimerkiksi, jos kysely on `SELECT email AS user_email FROM users` ja uudelleennimeä `email` `Email` taulukon oikeanpuoleisella paneelilla, sarakkeen tunnus olisi edelleen `user_email` Jos tämä sarake lisätään manuaalisesti käyttämällä oikealla puolella olevassa paneelissa olevaa `Lisää sarake -painiketta`, voit tarkistaa sen tunnuksen vasemmanpuoleisen paneelin `displayedData`-osiosta. Tyypillisesti, se on satunnaisesti luotu merkkijono alkaen `column`, kuten `column-320169e8-225a-482d-9853-ad34b3040220`.

**\<2> ** **`filterFn`** : suodattimen tyyppi täsmää. We support the following types: `equalTo` which is the same as `=` in SQL, `notEqualTo` which is the same as `<>` in SQL, `contains` which is the same as `LIKE '%value%'` in SQL, `does NotContain` which is the same as `NOT LIKE '%value%'` in SQL, `lessThan` which is the same as `<` in SQL, `notLessThan` which is the same as `>=` in SQL, `notMoreThan` which is the same as `<=` in SQL, `empty` which is the same as `IS NULL` in SQL, `notEmpty` which is the same as `IS NOT NULL` in SQL, `before` is used to compare time or date, which is the same as `>` in SQL, `after` is used to compare time or date, which is the same as `<` in SQL.

**\<3>** **`value`** : käytetään vertailevan arvon määrittämiseen.

**Toinen parametri:** kun suodattimen tila objekteja on useita, se määrittää, onko niillä "AND" suhde tai "OR"-suhde. Aseta se merkkijonoksi `and` "AND"-suhdetta varten ja `or` "OR"-suhdetta varten.

##### Tuetut komponentit

Taulukko

##### Esimerkki

```JavaScript
taulukko1.setFilters([{"id":"kk", "filterFn":"equalTo", "value":"May"}, {"id":"month", "filterFn":"equalTo", "value":"Kesäkuu"}], "or")
```

## Tallenna ILLA Drivelle

### Rakenne

```JavaScript
utils.saveToILLADrive({
fileName:string,
fileData:string,
fileType:string,
kansio:string,
allowAnonymous:boolean,
replace:boolean
})
```

### Kuvaus

Käytetään tiedoston tallentamiseen ILLA Driveen.

`fileName`: Tiedoston tunnisteen nimi ILLA Drivelle latauksen jälkeen.

`fileData`: Base64 tai raaka tiedostotiedot, kuten JSON, CSV, tavallinen teksti jne.

`fileType`: Yksi seuraavista arvoista: Auto, tavallinen teksti, JPEG, PNG, SVG, JSON, CSV, TSV, Excel(.xlsx). Kun käytät "Auto", yritämme määrittää tiedostotyypin, joka perustuu joko tiedostopäätteeseen tai tiedostoon.

`folder`: Kohdekansio tiedostojen lataamista varten. Jätä tyhjäksi ladataksesi pääkansioon. @ info: whatsthis

`allowAnonymous`: Käytetään sen määrittämiseen, onko anonyymi käyttöoikeus sallittu. Jos aiot asettaa hakemuksesi julkiseksi, anonyymi pääsy on otettava käyttöön. Jotta tietosi, tiedostot tai kansiot olisivat saatavilla anonyymisti, ne on sijoitettava anonyymiin kansioon.

![](https://illasoft.larksuite.com/space/api/box/stream/download/asynccode/?code=Y2U5MzRmOGMxYWE4YzI2MjQxMjZhOWQ3ODFjNDNkMjlfTDJkSzg0QktPUmlQa3VZcDUxYXBsZVZmVHVhazlXS1ZfVG9rZW46SnIyQ2JuaGpsb0s0dGJ4Z1libnVkSjZLc0lmXzE2OTgzMDQ1MTI6ODMwODExMl9WNA)

`replace`: Käytetään kun samassa hakemistossa on tiedostoja, joilla on sama nimi korvaa olemassa oleva tiedosto uudella tiedostolla tai nimeä uusi tiedosto automaattisesti uudelleen.
