#### setFiltre (array of objects, string)

Folosit pentru a seta filtre.

**Primul parametru** este o serie de obiecte. Fiecare obiect reprezintă o condiție de filtrare. Acesta include trei atribute:

**\<1>** **`id`** : Acesta se referă la id-ul coloanei. Dacă această coloană este rezultatul unei interogări a bazei de date, atunci id-ul ar fi numele coloanei pe care l-ați regăsit din interogare. De exemplu, în cazul în care interogarea este `SELECT e-mail AS user_email DE la utilizatori` şi redenumiţi `e-mail` ca `Email` folosind panoul din partea dreaptă a tabelului, ID-ul coloanei ar fi în continuare `user_email` dacă această coloană este adăugată manual folosind butonul `Add Column` din panoul din dreapta, poți să îi verifici id-ul în secțiunea `displayedData` din panoul din stânga. În mod obișnuit, va fi un șir generat aleatoriu începând cu `column` cum ar fi `column-320169e8-225a-482d-9853-ad34b3040`.

**\<2> ** **`filterFn`** : tipul filtrului se potrivește. Acceptăm următoarele tipuri: `equalTo` care este la fel ca `=` în SQL, `notEqualTo` care este la fel ca `<>` în SQL, `conţine` care este la fel ca `LIKE '%value%'' în SQL, `does NotContain" care este la fel ca `NOT LIKE '%value%'' în SQL, `lessThan`care este la fel ca`\<`în SQL,`notLessThan`care este la fel ca`>=`în SQL,`notMoreThan`care este la fel ca`\<=`în SQL,`goly`care este la fel ca`IS NULL`în SQL,`notEmpty`care este la fel ca`IS NULL`în SQL,`before`este folosit pentru a compara ora sau data, care este acelaşi cu`>`în SQL,`after`este folosit pentru a compara ora sau data, care este la fel ca`\<\` în SQL.

**\<3>** **`value`** : folosit pentru a seta valoarea de comparat.

**Al doilea parametru:** când sunt obiecte cu filtru multiplu, acesta determină dacă au o relaţie "AND" sau o relaţie "OR". Setaţi-l ca şir de caractere pentru o relaţie "AND", şi `sau` pentru o relaţie "OR".

##### Componente suportate

Tabel

##### Exemplu

```JavaScript
table1.setFilters([{"id":"lună", "filterFn":"equalTo", "value":"May"}, {"id":"lună", "filterFn":"equalTo", "value":"June"}], "sau")
```
