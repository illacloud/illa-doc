## <h2 hidden>About ILLA</h2>

Το ILLA σάς επιτρέπει να δημιουργείτε και να αναπτύσσετε γρήγορα εσωτερικές εφαρμογές. Συνδεθείτε με τις βάσεις δεδομένων σας και APIs, συγκεντρώστε UIs με σύρετε-και-πτώση οικοδομικά μπλοκ όπως πίνακες και μορφές, και γράψτε ερωτήματα για να αλληλεπιδράσετε με δεδομένα χρησιμοποιώντας SQL και JavaScript.

### What is ILLA?

ILLA is an open-source low-code platform for developers to build and deploy internal tools remarkably fast.

<img
  src='https://cdn.illacloud.com/official-website/img/docs/introduction/about-illa-cloud.png'
  alt=""
/>

Με την ILLA, μπορείτε εύκολα να συνδεθείτε σε διάφορες πηγές δεδομένων, όπως βάσεις δεδομένων, API / GraphQL endpoints, SaaS εργαλεία και υπηρεσίες αποθήκευσης σύννεφων. Αυτές περιλαμβάνουν δημοφιλείς επιλογές όπως PostgreSQL, MongoDB, MySQL, Stripe, Google Sheets, και AWS S3. Μόλις συνδεθεί, η ILLA μπορεί να τρέξει ερωτήματα για να μαζέψει και να ενημερώσει δεδομένα από αυτές τις πηγές. Το περιβάλλον εργασίας χρήστη (UI) παρέχει βολικά στοιχεία για την απεικόνιση και την τροποποίηση των δεδομένων, όπως πίνακες, διαγράμματα και μορφές.

Με την ILLA, μπορείτε γρήγορα και εύκολα να δημιουργήσετε ένα περιβάλλον εργασίας χρήστη (UI) χρησιμοποιώντας προ-χτισμένα widgets που μπορείτε να σύρετε και να πέσετε σε ένα καμβά στυλ πλέγματος. Αυτό απλοποιεί την ενσωμάτωση του front-end και του back-end της εφαρμογής σας και βελτιστοποιεί τη διαδικασία οικοδόμησης. Επιπλέον, η ILLA υποστηρίζει JavaScript μέσα σε widgets, ερωτήματα, και άλλα στοιχεία, επιτρέποντάς σας να προσθέσετε λογική, να μετατρέψει δεδομένα και να ορίσετε σύνθετες ροές εργασίας. Αυτό καθιστά εύκολο να προσαρμόσετε την εφαρμογή σας στις συγκεκριμένες ανάγκες σας.

### Get Start

Για να δοκιμάσετε το ILLA, ο πιο βολικός τρόπος είναι να εγγραφείτε και να συνδεθείτε στο [ILLA Cloud](https://cloud.illacloud.com/). 
Για να επισπεύσουμε τη διαδικασία αξιολόγησης, σας προσκαλούμε να συμμετάσχετε στο [Discord Community](https://discord.gg/illacloud), όπου μπορείτε να αποκτήσετε έναν κωδικό πρόσκλησης γρηγορότερα.

### Community

Για να λάβετε πιο έγκαιρη υποστήριξη κατά τη χρήση της ILLA, σκεφτείτε να γίνετε μέλος της [Discord Κοινότητας](https://discord.gg/illacloud). Αυτό θα σας επιτρέψει να λάβετε υποστήριξη απευθείας από την κοινότητα, η οποία μπορεί να σας βοηθήσει να επιλύσετε τυχόν ζητήματα πιο γρήγορα.
