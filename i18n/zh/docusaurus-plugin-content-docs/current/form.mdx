## <h2 hidden>是什么形式？</h2>

表单组件是一个视觉元素，代表任务或进程的进展或完成。 它显示一个根据指定进度值填充或缩减的水平栏。

### 属性

| 属性                            | 描述                                                                                                                                    |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| 事件处理程序                        | 检测和响应特定用户动作或事件，如点击、按键或表单提交。                                                                                                           |
| 已禁用                           | 非交互性且不能被用户修改或触发。                                                                                                                      |
| 禁用提交                          | 禁用表单提交功能。                                                                                                                             |
| 在提交时验证输入                      | 表单输入的正确性或完整性应在表单提交时进行验证。                                                                                                              |
| 成功提交后重置                       | 成功提交后自动重置其值和状态到其初始值或默认值                                                                                                               |
| 工具提示                          | 当用户悬停或与特定元素（如表单组件）交互时出现的小弹出式或信息信息。                                                                                                    |
| 显示标题                          | 表单组件是否应显示页眉或标题部分                                                                                                                      |
| 显示页脚                          | 表单组件是否应显示页脚部分                                                                                                                         |
| Hidden                        | 从用户视图中隐藏表单组件                                                                                                                          |
| 边框颜色                          | 围绕一个表单组件的边框颜色                                                                                                                         |
| 边框半径                          | 表格组件边框边角的曲线                                                                                                                           |
| Border Width                  | 窗体组件边框厚度或宽度                                                                                                                           |
| 二. 背景                         | 定义填充表单组件内容区域的颜色或图像                                                                                                                    |
| 阴影                            | 在表单组件中添加一个细胞，偏移“阴影”，从而使它在背景上显得略有提高或浮动效果                                                                                               |

### 方法

您可以使用其他组件来控制组件。 我们支持以下两种方法：

-   **提交**

提交填写表单的信息

-   **reset**

重置表单组件中的所有输入字段

-   **setValue** 值

To set the text area input value, for example, {{"value1"}}

| 属性     | 描述          |
| ------ | ----------- |
| 值      | Input value |

-   validate

验证输入信息是合法的

### 数据

The component has some commonly used data, which can be called via **`{{componentName.propertyName}}`** in the app.

| 属性名称                  | 描述                                                                                                                                      |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| 背景颜色                  | 指定表单组件的背景颜色                                                                                                                             |
| 边框颜色                  | 定义表单组件的边框颜色                                                                                                                             |
| borderWidth           | 决定表单组件周围边框的厚度或宽度                                                                                                                        |
| 已禁用                   | 是否禁用表单组件                                                                                                                                |
| 显示名称                  | 代表分配给表单组件的名称或标签                                                                                                                         |
| 页脚高度                  | 指定表单组件页脚部分的高度或大小                                                                                                                        |
| 格式数据                  | 持有用户在表单组件中输入的数据或值                                                                                                                       |
| 头部高度                  | 决定表单组件头部的高度或大小                                                                                                                          |
| 无效的                   | 表单组件的输入或值是否被认为无效。                                                                                                                       |
| radius                | 定义表单组件角的边框半径或曲线                                                                                                                         |
| 重置后续成功                | 表示一个表单组件应在成功提交后自动重置其值和状态到其初始值或默认值                                                                                                       |
| 阴影                    | 控制表单组件阴影效果的存在和外观                                                                                                                        |
| 显示页脚                  | 确定表单组件的页脚部分是否显示                                                                                                                         |
| 显示标题                  | 表单组件标题部分的可见性                                                                                                                            |
| 工具提示文本                | 持有与表单组件相关联的工具提示的文本或内容                                                                                                                   |
| 验证 InputOnSubmission  | 指定表单组件的输入值应该在提交表单时验证为正确性或完整性                                                                                                            |

Example: {{form1.disabled}}

### 使用大小写

我们将演示如何以输入的名称和年龄制作表单， 以及显示表单完成进度的进度栏。

#### 步骤 1 添加组件

添加 **`form`** 组件, 默认文本"Form " 和 "提交"按钮到画面。

添加 **`input`** 组件、 **`number input`** 组件、 **`bar progress`** 组件从`**插入**` 添加到表单中。

将输入组件标注为“名称”，将输入组件标注为“年龄”，将输入组件标注为“进度”。

将输入组件的占位符设置为“请将您的名字放在这里”，以及下面显示的数字输入的默认值。

![bar_layout](https://cdn.illacloud.com/official-website/img/official-site/components/bar_layout.jpg)

#### 步骤 2 配置组件

对于**`input`** 和 **`number input`** 组件, 我们可以将它们配置为**`bar progress`** 组件中的值, 以反映完成的进度。

1.  点击 **`input`** 和 **`number input`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择你想要更新的 **\`bar 进度** 组件作为事件的目标。
3.  选择 **`setValue` ** 动作并将其设置为

```jsx
{
  {
    input1.value && numberInput1.value
      ? "100"
      : input1.value || numberInput1.value
      ? "50"
      : "0";
  }
}
```

![bar_input_num_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_input_num_config.jpg)

注意：需要设置上述事件处理程序到 BOTH 输入和数字输入组件。

对于**`buton`** 组件，如果表单已完成，我们可以配置它来提交表单(e 条进化100%)

1.  点击 **`buton`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **`Control component` ** 正在执行中。 并选择 **`form`** 你想要更新的组件作为事件的目标。
3.  选择 **`submit`** 方法并将 `{{barProgress1.value == 100}` 放入**`仅在`** 时运行

![bar_button_submit_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_submit_config.jpg)

同样，当用户试图提交未完成的表单时，我们可以配置按钮来显示警告。

1.  点击 **`buton`** 组件来打开它的检查页面。 在 **`Event handler`**下单击`**+ 新**`。
2.  在编辑事件处理器中，选择 **\`显示通知** 在动作中，**`title`** "错误!" 和 **`description`** "请在提交前填写表单"。 Choose `Error` for **`type`**, put `{{2000}}` for the **`duration`** (2 seconds), and put `{{barProgress1.value<100}}` for **`Only run when`.**

![bar_button_show_config](https://cdn.illacloud.com/official-website/img/official-site/components/bar_button_show_config.jpg)
