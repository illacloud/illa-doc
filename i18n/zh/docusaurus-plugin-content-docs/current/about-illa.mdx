# 关于 ILLA

ILLA 使您能够快速构建和部署内部应用程序。连接到数据库和 API，使用拖放构建块（如表和表单）组装 UI，并使用 SQL 和 JavaScript 编写查询以与数据交互。

### 什么是 ILLA?

ILLA 是一个开源的低代码平台，供开发人员极为快速构建和部署的内部工具。

<img
  src="https://cdn.illacloud.com/official-website/img/docs/introduction/about-illa-cloud.png"
  alt=""
/>

使用 ILA，您可以轻松连接到各种数据源，如数据库、API/GraphQ Lendpoints、SaaS 工具和云存储服务。其中包括目前流行的，如 PostgreSQL、MongoDB、MySQL、Stripe、Google Sheets 和 AWS S3。一旦连接，ILLA 就可以运行查询来从这些源获取和更新数据。用户界面（UI）为可视化和修改数据（如表、图表和表单）提供了方便的组件。

使用 ILLA，您可以使用预先构建的小部件快速轻松地构建用户界面（UI），这些小部件可以拖放到网格样式的画布上。这简化了应用程序前端和后端的集成，并优化了构建过程。此外，ILLA 在小部件、查询和其他组件中支持 JavaScript，允许您添加逻辑、转换数据和定义复杂的工作流。这使您可以轻松地根据自己的特定需求自定义应用程序。
