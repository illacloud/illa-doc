# Button 按钮

## 什么是 Button?

Button组件允许用户通过单击按钮来触发动作或执行一个函数。

### Properties

| 属性          | 说明                                                         |
| ------------- | ------------------------------------------------------------ |
| Text          | 按钮显示的文本。                                               |
| Event Handler | 触发查询、控制组件或调用其他API以响应组件事件。              |
| Loading       | 组件是否应显示loading。                                      |
| Disabled      | 控制组件是否处于被禁用的状态。组件处于禁用状态时，无法对其进行修改或聚焦。 |
| Tooltip       | 用户可以在此处输入tooltip组件。tooltip将在聚焦时显示。支持Markdown格式。 |
| Hidden        | 动态控制组件是否隐藏。您可以通过动态布尔值更改隐藏状态。     |
| Variant       | 允许用户在按钮的两种不同样式之间进行选择：fill or outline。  |
| Theme Color   | 允许用户指定按钮的背景颜色和不透明度。                       |

### Event handler

| 事件  | 说明                                           |
| ----- | ---------------------------------------------- |
| Click | 当用户单击此按钮时，执行用户自定义的特定操作。 |

### Data

该组件有一些常用的数据，可以通过应用程序中的**`{{componentName.propertyName}}`** 调用这些数据。

| 属性名        | 说明                                      |
| ------------- | ----------------------------------------- |
| colorScheme   | 按钮的背景颜色                            |
| disabled      | 一个Boolean类型的值表示按钮是否为禁用状态 |
| displayName   | 组件的名称（即按钮1）                 |
| hidden        | 一个Boolean类型的值表示按钮是否为隐藏状态 |
| hiddenDynamic | 一个Boolean类型的值动态表示按钮是否为隐藏 |
| loading       | 是否显示loading                           |
| text          | 按钮上显示的文本                        |
| tooltipText   | tooltip的文本                             |
| variant       | 按钮的变体样式 (fill or outlined)         |



### 用例

接下来，我们将演示如何使用按钮来动态控制 **`form`** 表单组件。

#### 第一步 添加操作

我们创建了一个名为**`countries`**的表，包括5列：**`num_code`**,  **`alpha_2_code`**, **`alpha_3_code`**, **`en_short_name`**, **`nationality`**。我们用 **`en_short_name`**列中国家的名称作为选项的值。

然后，我们可以从操作列表中为Subabase创建一个新的操作，列出**`countries`**中的所有数据，并命名为supabasedb1

```sql
select distinct en_short_name as value from "Countries" order by value;
```

这是输出:

<img src='https://cdn.illacloud.com/official-website/img/docs/countries.png' alt='' />

单击**`Save`** 和**`Run`** 激活这个操作。

#### 第二步 添加组件

首先我们添加**`Form`**组件，标签为'Survey'。然后我们拖入一个input组件，一个select组件和一个number input组件到表单中，标签分别为**`Name`, `Nationality`,** 和**`Age`。**

然后，我们映射要选择组件的**`countries`** 值

1. 在映射模式下，将数据源设置为 `{{supabasedb1.data}}` 
2. 在映射选项中，将值设置为 `{{item.value}}`

<img src='https://cdn.illacloud.com/official-website/img/docs/button1.png' alt='' />

然后我们可以配置 **`Submit`** 按钮来控制**`Form`** 组件的 **`Send`** 方法。

1. 在 Edit event handler中, 从action中选择**`Control component`**操作 , 选择要更新为事件目标的**`form`**组件。
2. 选择**`Submit`** 操作。

<img src='https://cdn.illacloud.com/official-website/img/docs/countries2.png' alt='' />
